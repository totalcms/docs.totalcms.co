---
title: "Templates"
description: "Customize Total CMS login and download authentication pages using special Twig templates with built-in form variables and styling."
---
The following are special template that you can use to customize different parts of the CMS.

Until there is an official way to add templates, you can manually add these files into a `templates` folder
inside of your `tcms-data` folder.

### Login Page

* `login-above.twig`: content added <u>above</u> the login page form
* `login-below.twig`: content added <u>below</u> the login page form

### Download Auth Page

* `download-auth-above.twig`: content added <u>above</u> the download authentication page form
* `download-auth-below.twig`: content added <u>below</u> the download authentication page form
